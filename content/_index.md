## A code editor that paddles its own board

PaddleBoard is a fork of [Zed](https://zed.dev) tuned for AI-native development:
bring-your-own-key LLM providers, a built-in agent workbench, and an AI Dock for
managing agents, skills, and MCP servers — with telemetry hard-disabled by default.

### Built for you, not for a plan

- **Bring your own keys** — OpenAI, Anthropic, Google, Vertex AI, Bedrock, and more. No hosted plan, no upsells.
- **Agent frameworks** — first-class support for ADK, LangGraph, CrewAI, and AutoGen.
- **AI Dock** — install agents, skills, and MCP servers from one place.
- **Tiered languages** — common languages work out of the box; opt into the rest from *Manage Languages*.

> **Status: alpha.** Build from source for now — released binaries are on the way.
