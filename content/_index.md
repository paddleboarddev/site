## An open-source AI IDE

PaddleBoard is an open-source AI IDE: a fast, local-first code editor — built on [Zed](https://zed.dev) — with an agentic, AI-native workflow built in. The full source is on [GitHub](https://github.com/paddleboarddev/paddleboard) under the GPL. No account, no hosted plan, no telemetry.

## A harness for coding agents

Beyond editing, PaddleBoard is a harness for AI coding agents. Bring your own model keys — OpenAI, Anthropic, Google, Gemini Enterprise, Bedrock, Z.ai, Fireworks, and more — run a built-in agent workbench with delegation, and install or orchestrate agents, skills, and MCP servers from the AI Dock, complete with live logs and parallel runs.

## Private by default, and yours

Telemetry is hard-disabled at the source, and even spelling and grammar checking runs fully offline. You own the relationship with your model providers, and your code never leaves your machine unless you send it there.
